# Project Data

- CambodiaBombing.geojson: raw data for bombing of Cambodia
- cambodiaNightLights2025.tif: clipped nighttime lights data for Cambodia only
- LaosBombing.csv: raw bombing data for Laos only (extracted from THOR Vietnam Data)
- laosBombingClean: cleaned bombing data for Laos
- LaosBoundaryLvl1: province boundaries for Laos 
- laosNightLights2025.tif: nighttime lights data for Laos only (clipped from EOG global VIIRS raster)
- LaosSHDI.csv: raw subnational HDI data for Laos

I did minimal cleaning on the subnational HDI and Cambodia bombing data, so only the raw files are attached. 

The raw THOR Vietnam file is too large to be included in the repo. It can be downloaded here <https://data.world/datamil/vietnam-war-thor-data>. I used the full dataset (thor_data_vietnam.csv). I was unable to find clear documentation for the dataset. 

The raw nighttime lights data is also too large to be included. It can be downloaded here <https://eogdata.mines.edu/products/vnl/>. I used VNL v2.2 (median, masked). 

I could not find documentation for the Cambodia Bombing data, but some information can be found on the download page <https://data.opendevelopmentcambodia.net//dataset/us-bombing-in-cambodia-1965-1975->.
